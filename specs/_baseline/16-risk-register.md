# 16. Risk Register

## Scope

This register records the major risks visible in the active source snapshot and in the already created baseline documents. Graphify was used only as a navigation aid. Each row reflects a current concern, not a roadmap item or implementation plan.

## Scales

- Likelihood: `LOW`, `MEDIUM`, `HIGH`
- Impact: `LOW`, `MEDIUM`, `HIGH`, `CRITICAL`
- Severity: `LOW`, `MEDIUM`, `HIGH`, `CRITICAL`
- Detectability: `HIGH`, `MEDIUM`, `LOW`
- Reversibility: `HIGH`, `MEDIUM`, `LOW`
