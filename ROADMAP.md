- king election
- queen election
- police election

## v1.0

- release candidate
- full regression checklist green
- known bugs documented and stable
- architecture and rules kept in sync

## Ongoing Priorities

1. Keep the world stable before adding new mechanics.
2. Keep shared state authoritative on the server or database.
3. Keep documentation current as soon as a subsystem changes.
4. Keep regression risk low when touching `App.tsx` or `src/pocketbase.ts`.
